colorFromHex

  • colorFromHex(value: string): Color
    • Defined in api/draw/color.ts:121

    Creates a Color instance from the given hex color value.

    Valid hex values are:

    • FFFFFF (normal)
    • FFF (short)
    • FFFFFFFF (with alpha value)
    • FFFF (short with alpha value)

    The value is case insensitive (FFFFFF = ffffff).

    since

    0.0.1

    throws

    {IllegalColorValue} if the given hex color value is invalid

    Parameters

    • value: string

      a valid hex color value

    Returns Color

    the created color
