beginLine

  • beginLine(position: Point): void

  • Implementation of PolyLinePainter.beginLine

    • Defined in paint/painters.ts:279

    Start drawing a line on a canvas. Must be invoked before {@code drawTo} or {@code endLine}.

    Coordinates already added by the {@code coordinates} method are cleared before any painting is processed. The color set by the {@code borderColor} method will be used for the line.

    Parameters

    • position: Point

      the position on a canvas to begin drawing the line
