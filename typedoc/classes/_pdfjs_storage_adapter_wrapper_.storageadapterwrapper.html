    Loads the page overlays of the file by the given URI and the given {@code pageNumber}.

    All available StorageAdapter are used to perform this operation. The first adapter which resolves its data will be resolved by this operation. If any adapter makes usage of the {@link SkippableStorageAdapter}, the next adapter will be used.
