getThumbnails

  • getThumbnails(maxSize: number, ...pageNumbers: Array<number>): Observable<PageThumbnail>
    • Defined in api/document/pdf.document.ts:68

    Returns a observable which emits a PageThumbnail for each given page number. The observable completes when the last thumbnail is emitted.

    If the {@code maxSize} parameter is greater than the max width or height of the PDF page, the size of the PDF page will be used.

    Parameters

    • maxSize: number

      the max width or height of the thumbnails

    • Rest ...pageNumbers: Array<number>

      the page numbers for the thumbnails that should be returned

    Returns Observable<PageThumbnail>

    a observable which emits the thumbnails
